# CostBenefitAnalysisSkill

## Description

Systematically compares the costs and benefits of alternatives to support decision-making and resource allocation.

## Purpose

To provide a structured framework for evaluating options that ensures decisions are based on comprehensive consideration of all relevant factors.

## Methodology

1. **Alternative Identification**
   - Define the decision context and objectives
   - Identify all viable alternatives
   - Include "do nothing" as a baseline option

2. **Cost Identification**
   - Identify direct and indirect costs
   - Consider one-time and ongoing costs
   - Include tangible and intangible costs

3. **Benefit Identification**
   - Identify direct and indirect benefits
   - Consider short-term and long-term benefits
   - Include tangible and intangible benefits

4. **Quantification**
   - Assign monetary values where possible
   - Use estimates for difficult-to-quantify items
   - Apply appropriate discount rates for time value of money

5. **Analysis and Comparison**
   - Calculate net benefits (benefits - costs)
   - Compute benefit-cost ratios
   - Perform sensitivity analysis on key assumptions

## Output Guidelines

- **Alternatives Summary**: Description of options considered
- **Cost Breakdown**: Detailed cost analysis by category
- **Benefit Breakdown**: Detailed benefit analysis by category
- **Net Benefits**: Benefits minus costs for each alternative
- **Recommendation**: Preferred alternative with justification
- **Sensitivity Analysis**: How results change with key assumptions

## Example Usage

**Task**: Evaluate whether to upgrade legacy software to a modern platform.

**Application**:
1. Alternatives: upgrade, maintain legacy, rebuild from scratch
2. Costs: licensing, implementation, training, downtime, data migration
3. Benefits: improved efficiency, new features, reduced maintenance, better security
4. Quantify: implementation costs ($500K), annual savings ($200K), time horizon (5 years)
5. Analyze: NPV calculation, B/C ratio, sensitivity to adoption rate

## Best Practices

- Be comprehensive in identifying costs and benefits
- Use conservative estimates for benefits, realistic for costs
- Consider both quantitative and qualitative factors
- Apply appropriate time horizons for analysis
- Document assumptions and their rationale
- Perform sensitivity analysis on key variables
- Consider risk and uncertainty in the analysis
